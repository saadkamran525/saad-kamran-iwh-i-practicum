# Integrating With HubSpot I: Foundations Practicum

This project is a Node.js application built with Express, Axios, Dotenv, and Pug.  
It connects to HubSpot and creates/reads records from a custom object called Plants.

## Features

- View Plants records on the homepage
- Add a new Plant record from a form
- Redirect back to the homepage after creating a record

## Tech Stack

- Node.js
- Express
- Axios
- Dotenv
- Pug
